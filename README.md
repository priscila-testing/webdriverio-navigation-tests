# WebdriverIO Navigation Tests

Practice project for end-to-end (E2E) automated testing using **WebdriverIO** + Mocha.

## Features / Tests Included

- Basic navigation and assertions
- Handling iframes
- JavaScript alerts, confirms, and prompts
- Switching between windows/tabs
- Browser session management
- Basic HTTP Authentication (login with username & password)
- Taking screenshots of pages and specific elements

## Tech Stack

- WebdriverIO (latest)
- Mocha framework
- expect-webdriverio assertions
- Chrome browser

## How to Run

```sh
npm install
npx wdio run wdio.conf.js
```

This project is part of my QA Automation learning path and is intended for portfolio demonstration.
